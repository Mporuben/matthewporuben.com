## Build Setup

```bash
# install dependencies
$ npm i

# serve with hot reload at localhost:3000
$ npm run dev

# generate static project
$ npm run gen

# deploy
rsync -avH dist/ -e ssh root@matthewporuben.com:/path/to/file
```
